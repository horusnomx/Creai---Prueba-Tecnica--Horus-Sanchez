  },

    "tag": "A",
    "className": "latest_posts-card w-inline-block",

    "/resources/onboarding-is-where-trust-is-won-or-lost",
    "/resources/moving-beyond-ai-agent-washing"

}

Lectura:
  - cards_count > 0  → los artículos SÍ existen en el HTML
  - section_latest_posts.display == 'none' → el CSS oculta toda la sección
  - first_card.width_px / height_px = 0 → ni siquiera ocupan espacio en la página
  - Conclusión: contenido publicado en el CMS pero invisible para usuarios.
